## Runtime Impact

Adds one permission-checked API endpoint and a small service for creating pending metadata suggestions from validation reports. Existing ingestion, embedding, indexing, retrieval, and metadata acceptance behavior is unchanged.

## Database Impact

No database schema changes and no migrations. The bridge writes only new `document_metadata_suggestions` rows with `status=pending`; it does not modify `documents.metadata`.

## Risk Notes

The bridge uses existing metadata suggestion source enum values and records validation report provenance in the evidence excerpt. Unsupported fields are skipped conservatively. Suggestions generated by this bridge still require explicit reviewer acceptance before any production metadata or index state changes.
